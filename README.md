# Global E-Commerce Sales Analysis (Power BI Project)

## Project Overview

This project analyzes global e-commerce sales data using Power BI to uncover business insights and trends.
The dataset contains over 51,000 transactions across multiple regions, product categories, and customer segments.
An interactive Power BI dashboard was built to analyze sales performance, profitability, and regional market trends.


## Project Workflow

1. Data Cleaning – Prepared raw e-commerce dataset
2. Data Modeling – Built relationships and measures in Power BI
3. Data Visualization – Created interactive dashboards
4. Business Insights – Identified key trends in revenue, profit, and markets
5. Dashboard Sharing – Converted dashboard to PDF and interactive HTML report

   
## Key Metrics

Total Revenue: $12.6M  
Total Profit: $1.47M  
Total Orders: 25K+


## Key Business Insights

• Technology category generated the highest revenue  
• APAC market contributed the largest share of global sales  
• Tables sub-category showed negative profit due to heavy discounting  
• Strong seasonal sales growth observed in Q4
